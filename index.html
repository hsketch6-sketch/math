📘 Usage Guide

[Input Rules]
Use:
x^2 → power
xx → x²
xxx → x³

Auto-multiplication:
2x → 2*x

-------------------------

[Functions]

sin(x) → wave (periodic)
cos(x) → shifted wave
tan(x) → repeating with gaps

log(x) → ln(x), x>0
sqrt(x) → √x
abs(x) → |x|

Constant:
e = 2.718...

-------------------------

[Examples]

y=2x+1 → line
y=xx → parabola
y=1/x → hyperbola
y=sin(x) → wave

-------------------------

[Tips]

Combine freely:
y = sin(x)+x^2

Invalid domain:
log(x), sqrt(x)

Graph disappears?
→ value too large or undefined

📚 Graph Theory

[Linear]
y=ax+b
→ straight line
→ slope a
→ intercept b

[Proportional]
y=ax
→ passes (0,0)

[Quadratic]
y=ax²
→ parabola
→ has vertex

[Cubic]
y=ax³
→ S-shape

[Inverse]
y=1/x
→ hyperbola
→ asymptotes

[Trigonometric]

sin(x)
→ smooth wave
→ range [-1,1]

cos(x)
→ starts at 1

tan(x)
→ infinite jumps

[Log]
y=log(x)
→ x>0 only

[Sqrt]
y=√x
→ starts (0,0)

[Absolute]
y=|x|
→ V shape
