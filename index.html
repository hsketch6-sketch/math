📈 그래프 계산기

(x, y)

📘 사용법 + 한줄 설명

기본 함수
y=ax+b → 직선 (기울기와 절편)
y=ax → 원점을 지나는 직선
y=x^2 → 포물선 (위/아래로 볼록)
y=1/x → 쌍곡선 (축에 가까워짐)
y=sqrt(x) → 한쪽으로 시작하는 곡선
y=abs(x) → V자 그래프

지수 / 로그
y=2^x → 급격히 증가하는 곡선
y=e^x → 자연지수 증가
y=log(x) → 천천히 증가하는 곡선

삼각함수
y=sin(x) → 물결파
y=cos(x) → 위에서 시작하는 물결파
y=tan(x) → 끊어지는 S곡선

응용
y=1/(x^2+1) → 종 모양
y=x*e^x → 비대칭 급상승
y=sin(x)/x → 감쇠하는 파동

입력 팁
- x^2 또는 xx 가능
- sin, cos, tan 사용 가능
- 여러 그래프 추가 가능
- 삭제 버튼으로 제거
