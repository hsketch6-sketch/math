📈 그래프 계산기

📊 그래프 목록

📘 사용법

xx = x^2
xxx = x^3
^ 사용 가능 (x^2)

sin(x), cos(x), tan(x)
log(x), sqrt(x), abs(x)
2x → 자동으로 2*x 처리

예시:
y=2x+1  → 직선
y=xx    → 포물선
y=1/x   → 반비례
y=sin(x) → 사인함수

📚 함수 설명

y=ax → 정비례 (직선)
y=ax+b → 일차함수
y=ax^2 → 이차함수 (포물선)
y=1/x → 반비례 (쌍곡선)
y=sin(x) → 사인 (물결)
y=cos(x) → 코사인 (물결)
y=tan(x) → 탄젠트 (끊김)
y=log(x) → 로그
y=sqrt(x) → 루트
y=abs(x) → 절댓값 (V)
